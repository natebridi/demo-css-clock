# README

This is a simple CSS demo demonstrating several new (as of early 2026) CSS features, including carousels, custom functions, attr(), and anchor positioning. All this without any of the weird hacks (like hidden input elements) that have enabled interactive functionality in the past, nor convoluted CSS. 

Functioning clocks in several styles, navigable with an carousel, including pleasant animated tabs.

The only Javascript on the page is to initiatlize the clock times, but this could be inserted with any backend script. No scripting is involved in any of the functionality on the page.


## CSS features 
- [CSS Carousels](https://developer.mozilla.org/en-US/docs/Web/CSS/Guides/Overflow/Carousels)
- [Custom `@functions`](https://developer.mozilla.org/en-US/docs/Web/CSS/Guides/Custom_functions_and_mixins)
- [`attr()` used outside of `content` property](https://developer.mozilla.org/en-US/docs/Web/CSS/Reference/Values/attr)
- [Named anchor elements](https://developer.mozilla.org/en-US/docs/Web/CSS/Reference/Properties/anchor-name)
